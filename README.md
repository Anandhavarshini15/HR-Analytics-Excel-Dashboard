#  HR Analytics Excel Dashboard

## 📋 Project Overview
Analyzed IBM HR Employee Attrition data using 
Microsoft Excel to identify key factors causing 
employee attrition.

## Dataset Details
- Source: Kaggle — IBM HR Analytics Dataset
- Records: 1,470 employees
- Columns: 35
- Tool: Microsoft Excel

## Dashboard Features
- 5 KPI Cards (Total Employees, Attrition,
  Attrition Rate, Avg Salary, Gender Ratio)
- Attrition by Department Chart
- Gender Distribution Donut Chart
- Average Salary by Department Chart
- Overtime Impact on Attrition Chart
- Interactive Slicers (Department, Gender, Overtime)

##  Key Business Insights
1. Total Employees = 1,470
2. Attrition Rate = 16.12%
3. R&D has highest attrition — 133 employees
4. Overtime employees are 3x more likely to leave
5. Sales has highest average salary — $6,959
6. 60% Male vs 40% Female workforce

##  Tools & Features Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- COUNTIF, COUNTIFS, AVERAGEIF formulas
- Interactive Slicers
- IBM HR Analytics Dataset (Kaggle)
